Git の各種コマンド

git add
ローカルリポジトリに変更された情報を追加するつまり、変更されたワーキングツリーの一部をインデックスに追加する。
(これをステージングという)。
git commit
インデックス上に追加された変更情報にコメントをつけて( -m を付加)ローカルリポジトリに追加することができる。
git push
ローカルリポジトリ上にコミット（委任追加）されたファイルをリモートリポジトリに反映させることができる。
git branch
git branchだけのコマンドなら、現在のブランチが何かを確認することができる。
git branch ~　のように後ろに作りたいブランチを新たに作ることができる。
git merge
master ブランチが別のブランチの変更点を取り込むさいに使うコマンド
git log
現在 commit されている内容や push 可能なファイルを表示する。
